Install and use

Run atlas as a local binary with embedded SQLite

Every install surface exposes the same atlas command. The default database is sqlite://./.atlas/atlas.db, so indexing and review-context retrieval do not require a hosted service.

Latest release Homebrew tap npm package
Homebrew

macOS cask

brew install --cask dominic097/atlas/atlas
atlas version
npm

Node wrapper, atlas bin

npm install -g @dominic097/atlas
atlas version
Linux packages

amd64 and arm64 release assets

curl -LO https://github.com/dominic097/atlas/releases/download/v0.1.21/atlas_0.1.21_linux_amd64.tar.gz
tar -xzf atlas_0.1.21_linux_amd64.tar.gz
sudo install -m 0755 atlas /usr/local/bin/atlas
atlas version

Release assets also include Linux .deb, .rpm, and .apk packages for amd64 and arm64.

1

Index a repository

atlas index . --reindex

Builds the local symbol, call, route, and lexical graph into SQLite.

2

Retrieve bounded review context

atlas context --paths path/to/changed-file.go --query "review risk" --format json

Returns token-budgeted context around changed paths for review agents.

3

Expose MCP tools locally

atlas mcp --transport http --http 127.0.0.1:8765
atlas install skill --agent codex
atlas install skill --agent claude

Lets local assistant tools query Atlas for search, impact, and context without sending the repository graph to a server.
